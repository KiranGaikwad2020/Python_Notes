## Lists:
- Lists are used to store multiple items in a single variable.
- Lists are one of 4 built-in data types in Python used to store
collections of data.
- Lists are created using square brackets []
- Example:
    City = [Pune, Mumbai, Nagpur, Thane, Kolhapur]
